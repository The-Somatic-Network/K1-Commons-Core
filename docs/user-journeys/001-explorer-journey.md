# The Explorer — Discovery Journey

## Actor
A new user who has never interacted with the platform. They want to understand what the platform offers before committing time or money.

## Goal
Discover the value proposition and available services.

## Steps
1. Lands on the platform homepage.
2. Browses categories and featured co-ops.
3. Reads about the solidarity model and cooperative structure.
4. Creates an account after seeing enough value.

## Pain Points
- Information overload — too many categories with no clear guidance.
- Unclear value proposition — doesn't understand the co-op model.
- Trust barriers — hesitant to create an account without proof of legitimacy.

## Success Criteria
The user creates an account and follows or bookmarks at least one service listing.
