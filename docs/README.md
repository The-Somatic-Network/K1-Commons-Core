# Project Documentation

Central hub for all living documentation for the K1 Commons Core project.

## Structure

| Directory | Purpose |
|-----------|---------|
| `adr/` | Architecture Decision Records (ADRs) capturing key technical decisions |
| `architecture/` | High-level system architecture, tech stack, and component diagrams |
| `diagrams/` | SVG/Mermaid diagrams (system, sequence, ERD, deployment) |
| `user-journeys/` | User stories, flows, and experience maps |
| `templates/` | Reusable templates for ADRs, user journeys, etc. |

All docs here are **works in progress** and will evolve as the project is built out.
